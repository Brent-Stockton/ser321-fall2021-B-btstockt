You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /add?num1=3&num2=4 adds the two inputs and responses with the result
  • /subtract?num1=4&num2=3 subtracts the two inputs and responses with the result
  • /manbearbig takes you to a real life picture of the creature(extra just for fun)
